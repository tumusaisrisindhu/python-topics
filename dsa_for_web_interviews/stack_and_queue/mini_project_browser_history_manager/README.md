# Browser History Manager

A Python project demonstrating Stack concepts through a browser history system.

## Features

- Visit new pages
- Go back
- Go forward
- Track current page
- Display history

## Concepts Covered

- Stack
- LIFO
- Push
- Pop
- Undo/Redo mechanism

## Run

```bash
python app.py
```

## Files

history.py  -> Browser logic

app.py      -> Main program

notes.md    -> Revision notes
