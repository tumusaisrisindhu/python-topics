# Text Analyzer

A mini project built while learning Strings in Python.

## Concepts Used

- Strings
- Hashing
- Dictionaries
- Frequency Counting
- File Handling

## Features

- Read text file
- Count words
- Count characters
- Find most frequent words

## Run

```bash
python app.py
```

## Sample Output

```text
TEXT ANALYSIS
------------------------------
Total Words: 9
Total Characters: 56

Top 5 Frequent Words:
python: 3
is: 3
easy: 1
to: 1
learn.: 1
```
