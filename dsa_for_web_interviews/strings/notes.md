### Palindrome

A string that reads same forwards and backwards.

Example:

madam
racecar
level

Check:

```python
word == word[::-1]
```

---

### Frequency Counting

Count occurrence of each character.

Example:

apple

a → 1
p → 2
l → 1
e → 1

### Sliding Window

Used for substring problems.

Keeps a moving window.

Useful for:

- Longest substring
- Maximum sum subarray
- Distinct characters
