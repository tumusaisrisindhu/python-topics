# API Request Logger Decorator

## Overview

This project demonstrates how decorators are used to add logging functionality to API functions without modifying their code.

The decorator automatically:

- Records execution time
- Records function name
- Records arguments
- Stores logs in a file

---

## Files

### app.py

Contains API functions.

### logger_decorator.py

Contains the custom logging decorator.

### logs.txt

Stores execution logs.

---

## Run Project

```bash
python app.py
```

---

## Concepts Used

- Decorators
- Nested Functions
- Wrappers
- \*args and \*\*kwargs
- File Handling
- Execution Timing
